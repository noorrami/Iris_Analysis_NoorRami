# Exploring and Visualizing the Iris Dataset

## 📌 Project Overview
This project is part of my AI/ML Engineering Internship at **DevelopersHub Corporation**. The goal is to perform Exploratory Data Analysis (EDA) on the classic Iris dataset to understand data distributions and feature relationships.

## 📊 Dataset Information
The **Iris Dataset** consists of 150 samples of iris flowers from three different species:
* Setosa
* Versicolor
* Virginica

Each sample has four features: Sepal Length, Sepal Width, Petal Length, and Petal Width.

## 🛠️ Tools & Libraries
* **Python**: Base programming language.
* **Pandas**: For data manipulation and inspection.
* **Matplotlib & Seaborn**: For high-end data visualization.

## 🚀 Key Tasks Implemented
1. **Data Loading**: Loaded the dataset and inspected its shape and column names.
2. **Statistical Summary**: Used `.describe()` and `.info()` to extract insights like mean, standard deviation, and data types.
3. **Data Visualization**:
    * **Scatter Plots**: To visualize relationships between sepal and petal dimensions.
    * **Histograms**: To analyze the distribution of each feature.
    * **Box Plots**: To identify outliers across different species.

## 💡 Key Findings
* The **Setosa** species is clearly distinguishable from the others based on petal length and width.
* Features like petal length and petal width show a strong positive correlation.
* There are minimal outliers in the dataset, ensuring high data quality for future modeling.

## 📂 How to Run
1. Clone the repository.
2. Install dependencies: `pip install pandas seaborn matplotlib`.
3. Open `Iris_Analysis.ipynb` in Jupyter Notebook or Google Colab.
